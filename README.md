# Second

Second is a simple, secure TOTP (2FA) authenticator for iOS with optional iCloud sync.

## Features
- TOTP code generation
- QR code scanning
- Optional iCloud sync with end-to-end encryption
- Secure storage via iOS Keychain
- No ads, no tracking

## Requirements
- Xcode 15 or later
- iOS 16.0+
- Swift 5.9+

## Build
1. Open `Second.xcodeproj` in Xcode.
2. Select the `Second` target and a device/simulator.
3. Run.
